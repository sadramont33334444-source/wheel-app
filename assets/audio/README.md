# فایل‌های صوتی
این پوشه حاوی فایل‌های صوتی آفلاین است.

## فایل‌های مورد نیاز:
- `spin.mp3` - صدای چرخش گردونه
- `win.mp3` - صدای برد

## نکته مهم:
برای استفاده واقعی، فایل‌های صوتی مناسب را در این پوشه قرار دهید.
می‌توانید از سایت‌های رایگان مانند:
- freesound.org
- zapsplat.com
- mixkit.co

فایل‌های صوتی را دانلود کرده و با نام‌های بالا در این پوشه کپی کنید.

اگر فایل‌های صدا وجود نداشته باشند، برنامه بدون خطا اجرا می‌شود
ولی صدایی پخش نمی‌شود.
